# mdWorX palette reference

29 palettes shipped &middot; 17 dark &middot; 12 light.

Each card shows two states of the live editor:

- **Cursor elsewhere** &mdash; markers hidden / replaced (the rendered look).
- **Cursor on line** &mdash; raw markdown source revealed.

Every marker token is coloured via the palette accent; bold / italic / strike / inline-code each get their own palette-derived colour. See [`palette-reference.pdf`](palette-reference.pdf) for a printable version.
