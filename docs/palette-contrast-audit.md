## User-reported issue: Catppuccin Mocha

Catppuccin Mocha passes WCAG everywhere (`accent vs page = 8.1:1`), but the
user reports list markers and bullet characters are hard to see. This is a
perceptual issue not captured by WCAG luminance contrast: `#cba6f7` (mauve
accent) sits on `#1e1e2e` (dark purple-tinted page), and the hue similarity
makes the markers blend visually even though luminance differs.

WCAG measures luminance only and would say these colours pair fine.
Perceptual contrast models (APCA, color-opponency) would flag this.

## Fixes applied

- `.cm-md-list-number` font-weight bumped to `700` to match
  `.cm-md-bullet`. The thin glyph form of digits was a contributing
  factor for perceptual weakness in palettes where accent hue is close
  to page hue (especially Catppuccin Mocha).

## Outstanding (recommend revisiting palette default colours)

Light palettes have severe failures because the source theme spec uses
saturated mid-tone colours on near-white pages:

- **Ayu Light**: every heading + accent at 2.0â€“2.8:1. Source theme uses
  `#fa8d3e` (orange) for headings and accent. Recommend darkening to
  `#d65d0e` (`fa8d3e` shifted -20% lightness) for AA compliance.
- **Nord Light**: H2â€“H5 at 1.4â€“2.5:1. Aurora colours don't translate to a
  light bg cleanly. Recommend a darker Nord shade set.
- **Catppuccin Latte**: H2â€“H6 at 2.3â€“2.96:1. Latte palette colours are
  pastel-on-light; consider using Latte's darker text colours.
- **PLN Light**: every coloured role fails. Recommend reverting to a
  high-contrast set or accepting the design choice.

Dark palettes are mostly fine but flag:

- **PLN Dark**: accent / link / H1 all sub-4.5:1.
- **Solarized Dark**: link at 4.08:1 (just under AA).
- **Gruvbox Dark**: H1 at 2.69:1 (red on dark grey).

The light palettes are the bigger problem. Their original spec authors
chose colours for hue identity, not luminance accessibility. Each one
needs a per-heading darker variant for AA compliance, which we can
maintain in a separate "high-contrast variant" of the palette rather
than altering the canonical look.
